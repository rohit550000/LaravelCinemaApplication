<h1>How to run the code</h1>
<ul>
    <li>Clone repo or Download zip file and cd into it</li>
    <li><b>run</b> : composer install</li>
    <li><b>run</b> : php artisan serve</li>
    <li><b>Visit</b> to localhost:8000 in your browser</li>
</ul>
